==== DATABASE SETUP ====
1. Go to localhost/phpmyadmin
2. Create a new database named: "usccoop2024"
3. Go to navbar select -> Import
4. Import "usccoop2024.sql", click "OK" scroll below
5. Test the registration website at localhost/registration u should see data on the tables
